# File Search Tool

Ferramenta simples em Python para buscar uma palavra dentro de arquivos de uma pasta.

O objetivo do projeto é estudar lógica, manipulação de arquivos e organização de código,
servindo como base para evoluções futuras.

## Funcionalidades atuais
- Seleção de uma pasta no sistema
- Busca por palavra-chave em arquivos
- Exibição dos arquivos onde a palavra foi encontrada

## Tecnologias utilizadas
- Python
- Tkinter
